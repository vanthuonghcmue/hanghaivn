Ridhi Companion is a free companion plugin for Ridhi Theme.

== Description ==

Ridhi Companion is a plugin that adds extra functionalities and features to the Ridhi Theme by Rara Theme. The plugin when activated adds About, Client, Contact, Service, Social, Team and Testimonial sections to the theme. It also adds Header and Social Media configuration settings on the customizer.

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select ridhi-companion.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download ridhi-companion.zip
2. Extract the ridhi-companion directory to your computer
3. Upload the ridhi-companion directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard
